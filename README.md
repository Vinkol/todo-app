# Task Manager App

Простое React-приложение для управления задачами.

## 🚀 Функционал
- Добавление и удаление задач
- Отметка задач как выполненных
- Фильтрация (все / активные / выполненные)

## 🛠️ Технологии
- React
- JavaScript (ES6+)
- HTML / CSS
